# Changelog

## Changes Not Deployed

---

### Features

---

### Bug fixes

- fix: prevent infinite hot retry loop for stuck noop/expired transactions by adding max retry limit (10 attempts) and exponential backoff on send errors
- fix: add minimum 1 second delay on all queue processing errors to prevent CPU-intensive tight loops
- fix: correct error message format strings in ProcessPendingTransactionError and ProcessInmempoolTransactionError that were printing relayer ID instead of the actual RPC error
